# Entrada e Saída

- função _input_
~~~ py
nome = input("Informe o seu nome: ")
>>> Informe o seu nome:
~~~

- função _print_

~~~ py
nome = "Guilherme"
sobrenome = "Carvalho"

print(nome, sobrenome)
print(nome, sobrenome, end="...\n")
print(nome, sobrenome, sep="#")

>>> Guilherme Carvalho
>>> Guilherme Carvalho...
>>> Guilherme#Carvalho
~~~
